# PHP-Line-Bot
Simple Line Bot Framework based on Slim 3 <br>

## Usage
1. git clone <br>
2. composer update <br>
3. Edit your access_token, secret, and database configuration at .env <br>
4. Write your line bot code at app/MainEvent.php <br>
5. Publish :* <br>
